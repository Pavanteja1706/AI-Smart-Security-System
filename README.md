\# AI Smart Security Monitoring System



An AI-powered home security system built with Python, YOLOv8, OpenCV, and Face Recognition.



The system detects people in real-time, identifies known individuals, detects intruders, triggers alarms, sends email alerts, and provides a live monitoring dashboard accessible remotely.



\## Features



\- Real-time person detection using YOLOv8

\- Face recognition for known vs unknown individuals

\- Intruder detection with automatic image capture

\- Alarm alert when an unknown person is detected

\- Email notifications with captured evidence

\- Event logging system

\- Live monitoring dashboard using Flask

\- Remote access using ngrok



\## Technologies Used



\- Python

\- OpenCV

\- YOLOv8

\- face\_recognition (dlib)

\- Flask

\- SMTP Email Automation

\- Computer Vision

\- Deep Learning



\## System Workflow



Camera → Person Detection → Face Recognition → Intruder Detection → Alarm + Email Alert → Dashboard Update



\## How to Run



1\. Clone the repository



2\. Install dependencies



3\. Run the dashboard



4\. Open browser (http://127.0.0.1:5000)
